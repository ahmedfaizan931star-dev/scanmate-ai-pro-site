Privacy Policy

For ScanMate AI Pro, an offline-first Android document scanner app.

App name: ScanMate AI Pro

Developer / privacy contact: Faizan Ahmed — ahmedfaizan931star@gmail.com

Last updated: June 7, 2026

Plain-language summary: ScanMate AI Pro is designed as an offline-first scanner. Core scanning, OCR, document editing, and export features are intended to work locally on your device. Optional online Gemini AI features require your own API key and may send the text or prompts you choose to submit to Google’s Gemini service.

1. Data the app may handle

Depending on which features you use, ScanMate AI Pro may handle the following information on your device:

  • Scanned document images and imported images.
  • OCR text extracted from your documents.
  • Exported files such as PDF, DOCX, TXT, images, or ZIP backups.
  • Document names, page order, local document metadata, and app settings.
  • QR code content that you scan or generate.
  • An optional Gemini API key if you choose to enter one in Settings.

The core app does not require account creation, login, cloud sync, or a developer-operated cloud account to scan and manage documents.

2. Permissions and why they are used

  • Camera: used to scan documents and QR/barcode content when you choose to use camera features.
  • Network / Internet: used only for features that need a network connection, such as optional Gemini AI requests or opening external help links.
  • Biometric / fingerprint: used for device-level authentication where vault or protected access features are enabled and supported by the device.
  • Files and media access / system picker: used when you choose to import images/files or save/export documents.

Android may show runtime permission prompts before access is granted. You can manage permissions in Android system settings.

3. Local document processing

Core scanning, page cleanup, OCR history, export, QR tools, and file-management workflows are designed to operate locally on your device. Your scanned files are not automatically uploaded to developer-owned servers for core app functionality.

You control when documents are created, imported, exported, shared, or deleted.

4. Optional online Gemini AI features

ScanMate AI Pro may include optional AI tools that work only if you add your own Gemini API key and intentionally submit text or prompts. If you use those features, the content you submit, such as OCR text or a prompt, may be transmitted over the internet to Google’s Gemini API for processing.

The app does not require these AI features for core scanning, OCR, PDF export, QR tools, or local file management. If you do not add an API key or do not submit content to the AI tools, these optional requests are not made.

Third-party AI services may process submitted content under their own terms and privacy practices. Review the provider’s policy before sending sensitive documents.

5. Data sharing and sale

The developer does not sell personal or sensitive user data. The developer does not operate a required cloud sync server for the core document scanner features.

Data may leave your device only when you choose to share/export files, open external links, or use optional online AI features that require sending content to a third-party provider.

6. Data retention and deletion

Documents, OCR text, exports, QR history, app settings, and optional API keys remain on your device unless you delete them or uninstall/clear app data using Android settings.

You can delete local files and app data through the app where available or through Android system settings. Because ScanMate AI Pro does not require an account for core features, there is no developer-hosted account database to delete for normal offline use.

For help, contact ahmedfaizan931star@gmail.com.

7. Security

ScanMate AI Pro is designed to keep core document work local. Optional API keys are intended to be stored on the device using Android secure storage where available. Network requests for optional online AI features should use HTTPS.

No app can guarantee absolute security. Protect your phone with a screen lock, keep Android updated, and avoid sharing sensitive exported files with untrusted parties.

8. Children

ScanMate AI Pro is a general productivity tool and is not specifically directed to children. If a parent or guardian believes a child has provided personal information through optional external services, they can contact us for assistance.

9. Changes to this policy

This Privacy Policy may be updated when the app changes. The latest version will be posted on this page with a new “Last updated” date.
