Privacy Policy

App: ScanMate AI Pro · Developer: SynthByte Apps · Last updated: June 7, 2026

ScanMate AI Pro is an offline-first Android document scanner. This policy explains how the app handles documents, OCR text, permissions, optional AI features, and user data.

Information the app processes

The app may process documents, images, OCR text, QR/barcode content, file names, export settings, and app preferences that the user creates or imports. Core scanning and document management are designed to run locally on the user’s device.

Account and cloud sync

ScanMate AI Pro does not require a user account, login, or developer-operated cloud sync for core scanning features in the current version.

Permissions

Camera: used to scan documents and QR/barcodes. Files/media/storage: used to import images and save/export user-selected files. Biometric/security features: may be used for local vault or app-lock workflows when enabled by the user.

OCR and document processing

OCR and document processing are intended to work locally on the device where supported by the app version and Android device. The developer does not sell user documents or OCR text.

Optional AI features

Some AI tools may be optional and may require the user to provide an API key or choose a provider. If the user enables such features and sends selected content for AI processing, that content may be processed by the selected third-party AI provider according to that provider’s terms and privacy policy. Core scanning features are designed to work without enabling AI.

Data sharing

The developer does not sell personal or sensitive user data. The app does not require users to send documents to the developer for core scanning, OCR, editing, and export workflows.

Data retention and deletion

Documents and exported files remain on the user’s device unless the user deletes them, exports them, shares them, or stores them elsewhere. Users can delete documents inside the app where available, delete exported files using Android file tools, or clear app data from Android system settings.

Security

The app is designed with local-first document handling. Users should protect their device with a screen lock and avoid sharing sensitive exported files with untrusted parties.

Children

ScanMate AI Pro is a productivity tool and is not specifically directed to children.

Changes to this policy

This policy may be updated when app behavior changes. The latest version will be posted on this page.
