FAQ

Does ScanMate AI Pro require an account?

No account is required for core scanning, OCR, local file management, and export features.

Are my scanned documents uploaded to the developer?

Core scanning and document processing are designed to happen locally on your device. The developer does not operate a required cloud sync server for core features.

When is internet used?

Internet may be used for optional user-key Gemini AI features, opening external help links, or any feature you intentionally use that needs a network connection.
