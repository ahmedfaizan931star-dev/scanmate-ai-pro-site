Common fixes

  • If camera does not open, check Android camera permission for the app.
  • If imports fail, try using Android’s system file picker again and confirm the file is an image or supported document type.
  • If optional AI does not work, check internet connection, Gemini API key, and selected model.
  • If exported files are missing, check the app’s export folder and Android file permissions/settings.
